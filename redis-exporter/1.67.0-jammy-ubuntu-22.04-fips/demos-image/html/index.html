Demo Information

FIPS Compliance

This demo uses FIPS 140-3 validated cryptographic modules:

  • wolfSSL FIPS v5.8.2 (Certificate #4718)
  • golang-fips/go v1.25
  • wolfProvider for OpenSSL 3.x

Endpoints

Metrics: http://localhost:9121/metrics Prometheus: http://localhost:9090 Grafana: http://localhost:3000 Redis: localhost:6379
