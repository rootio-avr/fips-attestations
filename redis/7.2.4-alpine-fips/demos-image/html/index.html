🚀 Quick Start Commands

Run the demo tests:

docker run -it --rm redis-fips-demos:latest /opt/demos/scripts/persistence-test.sh

Start with persistence demo (default):

docker run -d -p 6379:6379 --name redis-demo redis-fips-demos:latest

Test connection:

docker exec redis-demo redis-cli PING

Verify FIPS compliance:

docker exec redis-demo fips-startup-check
